================
SBC(Jetson Nano)
================

This is a description of the Jetson Nano, the core of the Zetabot.

.. image:: ../images/sbc_1.jpg

.. list-table:: 
   :header-rows: 1
   
   * - Number
     - Interface
   * - 1
     - Heat Sink, Micro SD Card Terminal
   * - 2 
     - GPIO 40-pin
   * - 3
     - Micro 5-pin terminal, data
   * - 4
     - Internet port
   * - 5
     - USB 3.0 port
   * - 6,7
     - HDMI port
   * - 8
     - Power cable
   * - 9
     - Camera connector
